# ❤️ Heart Disease Risk Predictor

An AI-powered web application that predicts cardiovascular disease risk using Logistic Regression.

## 📌 Project Overview
This project uses machine learning to assess heart disease risk based on patient clinical data. 
The model is trained on the UCI Heart Disease dataset containing patients from Cleveland, Hungary, 
Switzerland, and VA Long Beach.

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit
- Plotly

## ⚙️ Data Preprocessing
- Missing value imputation (median/mode)
- One-Hot Encoding for categorical features
- Label Encoding for binary features
- Standard Scaling for normalization

## 🤖 ML Model
- **Algorithm:** Logistic Regression
- **Train/Test Split:** 80/20
- **Class Weight:** Balanced (handles imbalanced data)

## 📊 Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

## 🖥️ App Features
- Interactive patient data input form
- Real-time risk prediction
- Animated risk gauge (Plotly)
- Professional medical dashboard UI
